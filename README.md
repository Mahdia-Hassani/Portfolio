# 🌐 Mahdia Hassani - Portfolio

This is my personal portfolio website built to showcase my skills, projects, and services as a front-end developer and UI/UX designer.

🔗 **Live Demo:**
https://mahdia-hassani.github.io/Portfolio/

---

## 🚀 About The Project

This portfolio website is designed to present my work in a clean, modern, and responsive way. It includes sections like Home, Services, Projects, About, Blog, and Contact.

The goal of this project is to highlight my ability to build user-friendly and visually appealing websites.

---

## 🛠️ Built With

* HTML5
* CSS3
* Bootstrap 5
* JavaScript
* AOS (Animate On Scroll)

## ✨ Features
